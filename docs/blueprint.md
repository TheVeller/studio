# **App Name**: Alert Insights

## Core Features:

- Email Import: Import Google Alerts emails (paste or upload .eml files) with Google account authentication for multiple senders.
- Alert Parser: Parse email content to extract key information (title, snippet, source, link).
- Relevancy Scoring: Score alert relevancy based on keywords and context using AI; tool will decide when to incorporate information.
- Draft Response: Generate a short draft response using AI.
- Log Display: Display all alerts and relevant alerts in separate logs.

## Style Guidelines:

- Primary color: Clean white (#FFFFFF) for a simple, focused interface.
- Secondary color: Light gray (#E0E0E0) for subtle UI elements.
- Accent color: Teal (#008080) to highlight important actions and information.
- Font: 'Roboto' for clear and readable text throughout the application.
- Use simple, intuitive icons to represent actions and categories.
- Clean, single-column layout for easy navigation and focus.
- Subtle animations for loading states and feedback.
